# Pre-Installation

You will do MOST OF THE THINGS in `pwsh`, How do you do/open it? 

- Open `cmd`; open it by typing `cmd` in windows search, then Run as Administrator.

After that

- Type `pwsh`, then enter.

Now you should be in `pwsh`

## scoop

Now [`scoop`](https://scoop.sh/), you can click that link and follow the guide or install it by typing on `pwsh`:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
```
Just Copy text/code above and paste it on your `pwsh`, then enter.
